About Bootstrap

Features and Benefits

Bootstrap is one of the most powerful front-end frameworks. It was built by Twitter for mobile-first apps. The most outstanding feature of Bootstrap is the huge selection of additional templates, themes, and boilerplate code. With Bootstrap, programmers use simple HTML code to implement layout without CSS code, it improves the efficiency of website construction to a large extense.

Examples of Components

Badges

Badge is a very useful component. It adds extra information like count or label to any content.

Badges are similar to labels; the primary difference is that the corners are more rounded. Badges are mainly used to highlight new or unread items.

Example of badges:

Dropdowns

Dropdowns are used for displaying lists of links. It is a toggleable menu that allows the user to choose one value from a predefined list.

Dropdowns are built on a third party library, Popper, which provides dynamic positioning and viewport detection.

Example of dropdowns
Action Another action Something else here 



        <div class="dropdown">
            <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">Dropdown button</button>
          <div class="dropdown-menu" aria-labelledby="dropdownMenuButton">
            <a class="dropdown-item" href="#">Action</a>
            <a class="dropdown-item" href="#">Another action</a>
            <a class="dropdown-item" href="#">Something else here</a>
          </div>
        </div>
